## 开发模式

```bash
make dev    # go run ./cmd/backend
```

## 构建二进制

```bash
make build  # 输出 ./dist/bilirec-windows.exe（Windows）
```

支持平台：Linux amd64/arm64、Windows、Android (.so)。

## Android 库

需要 Android NDK 24+ 及 `./lib/<abi>/libffmpegkit.so`：

```bash
# Linux 主机
make android os=linux

# Windows 主机
make android os=windows
```

`NDK_HOME` 从 `ANDROID_NDK_LATEST_HOME`、`ANDROID_NDK_ROOT` 或 `ANDROID_NDK_HOME` 中选取。

输出：`dist/android/<abi>/libbilirec.so`（默认 `arm64-v8a` 与 `x86_64`）。

## 要求

- Go 1.25+
- FFmpeg + ffprobe（可选，本地 MP4 转换）
- Android NDK r24+（Android 构建）

预编译二进制见 [GitHub Releases](https://github.com/bilirec/bilirec/releases)。
