## 启用调试

```bash
export DEBUG=true
```

服务器启动时会在日志中打印一个临时十六进制令牌。

## pprof 性能分析

调试模式下在 `/debug/pprof` 挂载 pprof。该路由受保护：需在请求头 `Authorization` 中填入启动日志中的十六进制令牌。

实现位于 `internal/modules/rest/rest.go`。

## 本地开发

```bash
make dev    # go run ./cmd/backend
```
