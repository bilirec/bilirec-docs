Bilirec 由三个独立但紧密协作的项目组成，共同提供完整的 B 站直播录制体验。

## 三件套

<CardGrid>
  <LinkCard title="bilirec" href="https://github.com/bilirec/bilirec">
    **录制后端** — 提供 REST API、流录制引擎、文件管理与通知服务。可在 Linux、Windows、Docker 或 Android 上运行。
  </LinkCard>
  <LinkCard title="bilirec-web（PWA）" href="https://github.com/bilirec/bilirec-web">
    **Web 管理界面** — 线上地址 [app.bilirec.org](https://app.bilirec.org/)，支持 PWA 安装到桌面/手机主屏幕，用于管理录制任务、订阅房间与浏览文件。
  </LinkCard>
  <LinkCard title="bilirec-mobile（Android）" href="https://github.com/bilirec/bilirec-mobile">
    **Android 客户端** — 在手机上直接运行 bilirec 后端（内嵌 `libbilirec.so`），无需额外服务器即可录制直播。
  </LinkCard>
</CardGrid>

## 典型使用路径

### 路径一：自架服务器 + Web 界面

1. 在树莓派、NAS 或 PC 上[安装 bilirec 后端](/zh-cn/guides/installation/)
2. 打开 [app.bilirec.org](https://app.bilirec.org/) 并连接你的后端地址
3. 完成 B 站扫码[登录](/zh-cn/guides/quick-start/)
4. [订阅或录制](/zh-cn/guides/recording/)直播间

### 路径二：Android 手机独立运行

1. 从 [bilirec-mobile Releases](https://github.com/bilirec/bilirec-mobile/releases) 安装 APK
2. 在 App 内[启动后端](/zh-cn/guides/android/)
3. 点击「打开录制程序」进入 PWA 管理界面
4. 默认连接 `http://127.0.0.1:8080`（等同 `localhost`），直接点击「登录」即可

### 路径三：远程访问（FRP）

1. 在自架服务器上[启用 FRP](/zh-cn/guides/frp/)
2. 从日志获取公网地址（如 `https://abc123.tunnel.bilirec.org`）
3. 在 app.bilirec.org 登录界面的服务器地址栏填入公网地址

## 各端适用场景

| 场景 | 推荐方案 |
|------|---------|
| 树莓派 / NAS 24 小时录制 | bilirec 后端 + bilirec-web PWA |
| 外出时远程管理 | bilirec 后端 + FRP + bilirec-web PWA |
| 手机上随时录制 | bilirec-mobile |
| 脚本 / 自动化集成 | bilirec 后端 [REST API](/zh-cn/api/overview/) |
| 自定义 Android App | `libbilirec.so` — 见 [Android 库接口](/zh-cn/guides/android-library/) |

<Aside type="tip">
bilirec-web 是通用前端，可连接任何 bilirec 后端实例。你只需在登录界面填入后端地址即可，无需为每个后端单独部署前端。
</Aside>
