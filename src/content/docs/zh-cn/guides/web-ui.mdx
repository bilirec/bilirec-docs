bilirec 的 Web 管理界面由 [bilirec-web](https://github.com/bilirec/bilirec-web) 提供，线上地址为 [app.bilirec.org](https://app.bilirec.org/)。它是一个通用前端，可连接任何 bilirec 后端实例。

## 功能概览

| 功能 | 说明 |
|------|------|
| 录制管理 | 手动开始/停止录制，查看 `recording` / `recovering` / `idle` 状态与统计 |
| 直播订阅 | 订阅房间、配置自动录制与开播通知 |
| 网页推送 | 通过 Service Worker 接收开播等 Web Push 通知 |
| 文件浏览 | 浏览录制文件，支持流式下载与 MP4 预览 |
| 文件共享 | 生成带过期时间的分享链接 |
| 格式转换 | 提交与管理 FLV/TS/FMP4 ↔ MP4 转换任务 |
| 磁盘监测 | 实时磁盘占用与分项统计 |
| 角色控制 | 管理员 / 查看者（查看者为只读界面） |
| 深色模式 | 浅色/深色主题，支持跟随系统 |
| 多语言 | 简体中文 / 繁体中文 |
| 多服务器 | 登录页可配置后端地址，连接不同 bilirec 实例 |
| PWA | 可安装到桌面或手机主屏幕，获得接近原生 App 的体验 |

<Aside type="tip">
bilirec-web 需配合正在运行的 [bilirec](https://github.com/bilirec/bilirec) 后端使用。也可[自行部署前端](/zh-cn/development/web/)以连接局域网 HTTP 后端或使用自定义域名。
</Aside>

## 访问 Web 界面

1. 确保 bilirec 后端已启动（默认端口为 `8080`，`HOST` 默认为空值即监听所有网卡）
2. 打开 [app.bilirec.org](https://app.bilirec.org/)

<Tabs>
<TabItem label="桌面端">

![桌面端登录界面](/images/web-ui/desktop-login.png)

</TabItem>
<TabItem label="移动端">

![移动端登录界面](/images/web-ui/mobile-login.png)

</TabItem>
</Tabs>

3. 在登录界面填入后端地址（本地使用可留空或填 `http://localhost:8080`）
4. 若设置了 `USERNAME` 和 `PASSWORD`，输入账号密码登录；未设置则直接进入

## 连接自架后端

[app.bilirec.org](https://app.bilirec.org/) 通过 **HTTPS** 提供，而多数自架 bilirec 后端默认只提供 **HTTP**。浏览器出于混合内容（Mixed Content）安全策略，**禁止 HTTPS 页面请求 HTTP 的局域网地址**（如 `http://192.168.x.x:8080`），**仅允许**连接本机：

- `http://localhost:8080`
- `http://127.0.0.1:8080`

| 后端位置 | 能否用 app.bilirec.org 连接 |
|---------|---------------------------|
| 本机（bilirec-mobile、PC 本机后端） | 可以，填 `localhost` 或 `127.0.0.1` |
| 局域网另一台设备上的 HTTP 后端 | **不能** — HTTPS 前端无法直连 HTTP 局域网地址 |
| 公网 HTTPS（FRP、自架域名、反向代理等） | 可以，填 `https://你的后端地址`；需 `PRODUCTION_MODE=true`，见 [常见问题](/zh-cn/guides/faq/#用-appbilirecorg-连远程后端一直提示会话过期) |

远程访问局域网内的 HTTP 后端，可考虑 [FRP](/zh-cn/guides/frp/) 等提供 **HTTPS** 的方案。自架 HTTPS 后端（不限于 FRP）通过 app.bilirec.org 登录时，同样建议开启生产模式。详见 [常见问题 — Web 管理界面](/zh-cn/guides/faq/#web-管理界面)。

## 安装 PWA

bilirec-web 支持 PWA（Progressive Web App），可安装到桌面或手机主屏幕，获得接近原生 App 的体验。安装后会记住上次连接的后端地址。

<Tabs>
<TabItem label="Android Chrome">

若浏览器检测到可安装，页面顶部会出现安装横幅，点击 **安装** 即可：

<img src="/images/web-ui/pwa-install-banner.png" alt="Android Chrome PWA 安装横幅" width="400" style="max-width:100%;height:auto;display:block;margin:0 auto" />

也可通过菜单 **⋮**，选择 **新增至主屏幕**（部分机型显示为「安装应用」）：

<img src="/images/web-ui/pwa-menu-add-home.png" alt="Chrome 菜单「新增至主屏幕」" width="400" style="max-width:100%;height:auto;display:block;margin:0 auto" />

在确认对话框中点击 **安装**：

<img src="/images/web-ui/pwa-install-dialog.png" alt="PWA 安装确认对话框" width="400" style="max-width:100%;height:auto;display:block;margin:0 auto" />

</TabItem>
<TabItem label="桌面浏览器（Chrome / Edge）">

打开 [app.bilirec.org](https://app.bilirec.org/) 后，点击地址栏右侧的 **安装** 图标：

<img src="/images/web-ui/pwa-desktop-login.png" alt="桌面 Chrome 登录页与地址栏安装按钮" style="max-width:100%;height:auto;display:block;margin:0 auto" />

在「安装应用程序」对话框中点击 **安装**：

<img src="/images/web-ui/pwa-desktop-install-dialog.png" alt="桌面 Chrome PWA 安装确认对话框" style="max-width:100%;height:auto;display:block;margin:0 auto" />

若地址栏没有安装图标，可打开菜单 **⋮**，选择 **安装 Bilibili 录制管理**（或「安装 bilirec」）。

</TabItem>
<TabItem label="iOS Safari">

点 **分享**，再选 **添加到主屏幕**。

</TabItem>
</Tabs>
