bilirec 的 Web 管理界面由 [bilirec-web](https://github.com/bilirec/bilirec-web) 提供，线上地址为 [app.bilirec.org](https://app.bilirec.org/)。它是一个通用前端，可连接任何 bilirec 后端实例。

## 访问 Web 界面

1. 确保 bilirec 后端已启动（默认端口为 `8080`，`HOST` 默认为空值即监听所有网卡）
2. 打开 [app.bilirec.org](https://app.bilirec.org/)

3. 在登录界面填入后端地址（本地使用可留空或填 `http://localhost:8080`）
4. 若设置了 `USERNAME` 和 `PASSWORD`，输入账号密码登录；未设置则直接进入

## B 站登录

进入主界面后，点击「登录」，再点击右上角头像按钮完成 B 站扫码登录。

## 连接自架后端

若后端不在本机运行，在登录界面的**服务器地址**栏填入后端 URL，例如：

- 局域网：`http://192.168.1.100:8080`
- FRP 公网：`https://abc123.tunnel.bilirec.org`

## 安装 PWA

bilirec-web 支持 PWA（Progressive Web App），可安装到桌面或手机主屏幕，获得接近原生 App 的体验。

**桌面浏览器（Chrome / Edge）：**
- 地址栏右侧点击「安装」图标，或菜单 →「安装 bilirec」

**手机浏览器：**
- iOS Safari：分享 →「添加到主屏幕」
- Android Chrome：菜单 →「安装应用」或「添加到主屏幕」

安装后，PWA 会记住上次连接的后端地址，下次打开直接进入管理界面。

</Aside>
