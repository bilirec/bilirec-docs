大陆版国产手机系统（华为 / 鸿蒙、OPPO / realme / OnePlus 的 ColorOS、小米 / Redmi 的 HyperOS、vivo / iQOO 的 OriginOS 等）对后台应用的管理比一般 Android 更严格。若未完成适配设置，Bilirec 在后台时可能出现：

- 网络请求被延迟或中断，导致开播检测、录制拉流超时；
- 前台服务的定时检查（约每 15 秒一次）被系统挂起，通知栏状态不再更新；
- 锁屏、切到其他 App，或从多任务界面划掉卡片后，服务被系统清理。

<Aside type="note">
本文针对上述**大陆国产系统**。其他品牌（如 Pixel、三星国际版等）通常只需按 [Android App](/zh-cn/guides/android/) 中的电池优化提示操作即可。
</Aside>

## 划掉多任务卡片会立刻终止应用

即使已在 App 内关闭电池优化，在多数大陆国产系统上，从多任务界面**手动上划划掉卡片**，仍属于最高优先级的「用户主动清理」——系统会**立即终止**该应用的后台进程。

关闭电池优化只能减少系统在卡片**仍留存**时的自动清理，**无法**防止您主动划掉卡片。若希望 Bilirec 在清理多任务后仍保持运行，通常需要**锁定多任务卡片**（各品牌操作见下方对应标签页）。

## 如何设置

请先完成 Bilirec App 内的**电池优化豁免**，再切换到您手机对应的系统标签页，按顺序**逐步**调整。

<Aside type="tip">
**不必一次做完所有步骤。** 每完成一步，先测试 Bilirec 是否已恢复正常；若问题已解决，**即可停止，无需继续往下做**。这样可以在解决问题的同时，尽量减少对手机省电策略的影响。
</Aside>

<Tabs>
<TabItem label="华为 / 鸿蒙">

### 第一步：关闭「应用自动管理」

主要解决：App 切到后台或重启后服务无法自启、被系统清理。

1. 打开手机 **设置**，点击 **电池**。
2. 点击 **应用启动管理**。
3. 在列表中找到 **Bilirec**，关闭其右侧的 **自动管理** 开关。
4. 在弹出的手动管理窗口中，**同时勾选** 以下三项：
   - **允许自启动**
   - **允许关联启动**
   - **允许后台活动**
5. 点击 **确定** 保存。

完成后请测试 Bilirec。若后台运行已正常，无需继续下一步。

### 第二步：加入电池优化白名单

主要解决：系统省电策略仍限制 Bilirec 后台活动。

1. 打开手机 **设置**，在顶部搜索栏输入 **电池优化** 并进入。
2. 点击顶部的下拉菜单，将 **不允许** 切换为 **所有应用**。
3. 在列表中找到 **Bilirec**，将其设为 **不允许**（即：不允许系统对本应用进行省电优化）。

<Aside type="tip">
不同系统版本的文案可能略有差异。关键是让 Bilirec **不受电池优化**，与 App 内弹窗提示的目标一致。
</Aside>

完成后请测试 Bilirec。若问题已解决，无需继续下一步。

### 第三步：休眠时保持网络连接

主要解决：锁屏或手机休眠后网络被切断，导致通讯中断或超时。

1. 打开手机 **设置**，点击 **电池**。
2. 点击 **更多电池设置**。
3. 找到 **休眠时始终保持网络连接**（部分版本显示为 **休眠时保持网络连接**），将其 **开启**。

完成后请测试 Bilirec（建议锁屏等待几分钟后再观察）。若问题已解决，无需继续下一步。

### 第四步：在多任务界面锁定应用

主要解决：从多任务列表一键清理时 Bilirec 被连带杀掉。

1. 从屏幕底部向上滑动并停顿，进入 **多任务管理界面**。
2. 找到 Bilirec 的卡片，用手指按住卡片并 **向下滑动**。
3. 当卡片右上角出现 **小锁头** 图标时，表示后台锁定成功。

### 使用移动数据时的额外说明

若在外出时使用 **移动数据**（5G / 4G）仍遇到连接卡顿、请求超时，可能是华为在使用移动数据时对后台进程施加了更严格的限制。可暂时连接 Wi‑Fi 或热点；在 Wi‑Fi 下限制通常会明显放宽。

若前几步已解决问题，其余步骤可不做；若仍有超时，再按顺序尝试后续步骤，或暂时改用 Wi‑Fi。

</TabItem>
<TabItem label="OPPO / ColorOS">

适用于大陆版 **OPPO**、**realme**、**OnePlus**（ColorOS 系统）。

### 第一步：关闭「应用自动管理」

主要解决：App 切到后台或重启后服务无法自启、被系统清理。

1. 打开手机 **设置**，点击 **应用**。
2. 点击 **应用启动管理**。
3. 找到 **Bilirec**，改为 **手动管理**。
4. 勾选 **允许自启动** 和 **允许后台活动**。

完成后请测试 Bilirec。若后台运行已正常，无需继续下一步。

### 第二步：加入电池优化白名单

主要解决：系统省电策略仍限制 Bilirec 后台活动。

1. 打开手机 **设置**，点击 **电池**。
2. 点击 **更多设置**。
3. 点击 **应用耗电管理**。
4. 找到 **Bilirec**，将其设为 **允许完全后台行为**。

<Aside type="tip">
不同系统版本的文案可能略有差异。关键是让 Bilirec **不受省电限制**，与 App 内弹窗提示的目标一致。
</Aside>

完成后请测试 Bilirec。若问题已解决，无需继续下一步。

### 第三步：关闭休眠时的系统自动清理

主要解决：夜间或熄屏后系统强制清理后台，导致通讯中断或服务被终止。

1. 打开手机 **设置**，点击 **电池**。
2. 点击 **更多设置**。
3. 关闭 **睡眠模式优化** 或 **夜间超长续航**（视系统版本显示项而定）。

完成后请测试 Bilirec（建议锁屏等待几分钟后再观察）。若问题已解决，无需继续下一步。

### 第四步：在多任务界面锁定应用

主要解决：从多任务列表划掉卡片时 Bilirec 被立刻终止。

1. 从屏幕底部向上滑动并停顿，进入 **多任务管理界面**。
2. 找到 Bilirec 的卡片，点击卡片右上角的 **⋮**，或 **长按** 卡片。
3. 选择 **锁定**。锁定后，即使点击「全清除」或手动上划，该卡片也会留在后台不被关闭。
