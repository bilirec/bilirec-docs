完成[安装](/zh-cn/guides/installation/)后，按以下步骤启动服务并开始使用。

## 启动服务

<Tabs>
<TabItem label="Linux">

```bash
./bilirec-linux-amd64
```

首次运行二进制时，程序会在可执行文件同目录生成 `.env` 模板文件，你可以直接编辑它修改配置。

</TabItem>

<TabItem label="Windows">

下载并运行 `bilirec-windows.exe`（双击或在 PowerShell 中执行）：

```powershell
.\bilirec-windows.exe
```

首次运行二进制时，程序会在可执行文件同目录生成 `.env` 模板文件，你可以直接编辑它修改配置。

</TabItem>

<TabItem label="Docker">

```bash
docker pull eric1008818/bilirec:latest
docker run -d \
  --name bilirec \
  -p 8080:8080 \
  -v /path/to/records:/app/records \
  -v /path/to/secrets:/app/secrets \
  -v /path/to/database:/app/database \
  eric1008818/bilirec:latest
```

更多配置（环境变量、卷挂载说明、测试版本标签）见 [Docker 指南](/zh-cn/guides/docker/)。

</TabItem>

<TabItem label="Android（bilirec-mobile）">

安装 [bilirec-mobile](https://github.com/bilirec/bilirec-mobile/releases) 后，打开 App 点击「启动」即可先启动后端服务。

详细说明见 [Android App](/zh-cn/guides/android/)。

</TabItem>
</Tabs>

## 进入 Web 界面

<Tabs>
<TabItem label="Linux / Windows">

1. 打开 [app.bilirec.org](https://app.bilirec.org/)。
2. 在登录页填入后端地址。由于浏览器安全限制，从 HTTPS 线上前端**只能**连接本机 HTTP 后端（`http://localhost:8080` 或 `http://127.0.0.1:8080`）；远程后端须为 HTTPS（如 [FRP](/zh-cn/guides/frp/) 隧道）。
3. 若配置了 `USERNAME` / `PASSWORD`，先完成账号登录。

</TabItem>

<TabItem label="Android（bilirec-mobile）">

1. 在 bilirec-mobile 内点击「打开录制程序」进入 PWA。
2. 如未自动跳转，也可手动打开 [app.bilirec.org](https://app.bilirec.org/)。
3. 一般情况下后端地址已是 `http://127.0.0.1:8080`，可直接继续下一步。

</TabItem>
</Tabs>

## B 站登录模式

<Tabs>

<TabItem label="controller（默认）">

服务启动后，打开 [app.bilirec.org](https://app.bilirec.org/)（或你配置的 `FRONTEND_URL`），点击「登录」，再到右上角点击头像按钮完成 B 站扫码登录。

```bash
export BILIBILI_LOGIN_MODE=controller # 默認已經是
```

<Tabs>
<TabItem label="桌面端">

![桌面端B站扫码登录界面](/images/web-ui/desktop-b-qrcode.png)

</TabItem>
<TabItem label="移动端">

![移动端B站扫码登录界面](/images/web-ui/mobile-b-qrcode.png)

</TabItem>
</Tabs>

</TabItem>

<TabItem label="startup">

启动时在终端显示二维码，用手机 B 站 App 扫码登录：

```bash
export BILIBILI_LOGIN_MODE=startup
```

</TabItem>

<TabItem label="anonymous">

直接匿名登录，无需扫码。未登录可能无法录制 1080p 及以上画质：

```bash
export BILIBILI_LOGIN_MODE=anonymous
```

</TabItem>

</Tabs>

## 使用方式

<Tabs>
<TabItem label="Web 界面">

1. 访问 [app.bilirec.org](https://app.bilirec.org/) 并连接后端。
2. 在「录制管理」里手动开始/停止录制。
3. 在「订阅管理」里添加房间并配置自动录制。
4. 在「录制文件」里浏览、下载或预览录制结果。

![桌面端登录界面](/images/web-ui/desktop-login.png)
![桌面端录制管理](/images/web-ui/desktop-recording.png)

完整说明见 [Web 界面（PWA）](/zh-cn/guides/web-ui/)。

</TabItem>

<TabItem label="API">

可通过 REST API 直接接入脚本或自动化流程：

```bash
# 启动录制
curl -X POST "http://127.0.0.1:8080/record/12345/start"

# 停止录制
curl -X POST "http://127.0.0.1:8080/record/12345/stop"
```

接口总览见 [API 概述](/zh-cn/api/overview/)。

</TabItem>
</Tabs>

## 配置文件加载顺序

1. 当前工作目录下的 `.env.local`（若存在，优先）
2. 可执行文件同目录的 `.env`（首次运行自动生成）
3. 环境变量（覆盖文件中的值）
4. Docker 环境：仅使用环境变量，不加载 `.env`

详见[配置概述](/zh-cn/configuration/overview/)。
