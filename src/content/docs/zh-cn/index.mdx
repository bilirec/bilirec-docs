## 核心能力

- **录制引擎稳定**：完整支持 HTTP-FLV / HLS-TS / HLS-fMP4，自动处理流中断与时间戳异常。  
- **录制策略灵活**：支持手动录制、开播自动录制、定时停止与无限时长录制。  
- **存储管理友好**：支持文件浏览、批量删除、浏览器内预览 MP4。  
- **UI 简单易用**：提供直观的 Web 管理界面，新手也能快速完成登录、订阅与录制管理。  
- **手机端适配（PWA）**：支持以 PWA 形式安装到手机主屏，移动端访问与操作体验更友好。  
- **接入方式完整**：REST API、Web 界面（PWA）、Android App 多端协同。  

## 3 分钟上手路径

1. 先看 [安装说明](/zh-cn/guides/installation/) 部署 bilirec。  
2. 按 [快速开始](/zh-cn/guides/quick-start/) 添加房间并启动录制。  
3. 进阶调优请看 [配置与调优](/zh-cn/configuration/overview/) 与 [I/O 调优](/zh-cn/configuration/io-tuning/)。
