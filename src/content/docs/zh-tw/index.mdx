## 核心能力

- **錄製與畫質**：完整支持常見直播流格式；畫質切換時會自動分段，減少壞檔案。錄 **4K / 杜比** 時根據畫質自動加大緩衝，無需手動改基礎設定。
- **錄製策略靈活**：支持手動錄製、開播自動錄製、定時停止與無限時長錄製。  
- **儲存管理友好**：支持檔案瀏覽、批量刪除、瀏覽器內預覽 MP4。  
- **UI 簡單易用**：提供直觀的 Web 管理介面，新手也能快速完成登入、訂閱與錄製管理。  
- **手機端適配（PWA）**：支持以 PWA 形式安裝到手機主屏，移動端存取與操作體驗更友好。  
- **接入方式完整**：REST API、Web 介面（PWA）、Android App 多端協同。  

## 3 分鐘上手路徑

1. 先看 [安裝說明](/zh-tw/guides/installation/) 部署 bilirec。  
2. 按 [快速開始](/zh-tw/guides/quick-start/) 添加房間並啟動錄製。  
3. 進階調優請看 [設定與調優](/zh-tw/configuration/overview/)、[I/O 調優](/zh-tw/configuration/io-tuning/) 與 [高碼率 / 4K 調參](/zh-tw/configuration/high-bitrate-tuning/)。
