bilirec 內置 FRP 客戶端，無需公網 IP 即可在外網安全存取後端管理任務與檔案。

## 啟用 FRP

最簡單的方式——使用官方免費公共服務：

```bash
export FRP_ENABLED=true
```

僅設置 `FRP_ENABLED=true` 即可，程式會自動使用 bilirec.org 官方 FRP 服務。

## 操作流程

1. **啟動後端**並確認 FRP 已啟用
2. **查看日誌**，找到公網存取地址，例如：

   ```
   FRP enabled in official-public mode
   https://abc1234567.tunnel.bilirec.org
   ```

## 自建 FRP 伺服器

若你有自己的 FRP 服務，可自定義設定：

```bash
export FRP_ENABLED=true
export FRP_SERVER=your-frp-ip:7000
export FRP_TOKEN=your-token
export FRP_BASE_DOMAIN=your-domain.com
```

啟動日誌會顯示 `FRP enabled in custom-selfhost mode`。

技術細節、協議約束與完整參數說明見 [FRP 設定](/zh-tw/configuration/frp/)。
