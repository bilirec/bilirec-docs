大陸版國產手機系統（華為 / 鴻蒙、OPPO / realme / OnePlus 的 ColorOS、小米 / Redmi 的 HyperOS、vivo / iQOO 的 OriginOS 等）對後台應用的管理比一般 Android 更嚴格。若未完成適配設定，Bilirec 在後台時可能出現：

- 網路請求被延遲或中斷，導致開播檢測、錄製拉流超時；
- 前臺服務的定時檢查（約每 15 秒一次）被系統掛起，通知欄狀態不再更新；
- 鎖屏、切到其他 App，或從多任務介面劃掉卡片後，服務被系統清理。

<Aside type="note">
本文針對上述**大陸國產系統**。其他品牌（如 Pixel、三星國際版等）通常只需按 [Android App](/zh-tw/guides/android/) 中的電池優化提示操作即可。
</Aside>

## 劃掉多任務卡片會立刻終止應用

即使已在 App 內關閉電池優化，在多數大陸國產系統上，從多任務介面**手動上劃劃掉卡片**，仍屬於最高優先級的「用戶主動清理」——系統會**立即終止**該應用的後台進程。

關閉電池優化只能減少系統在卡片**仍留存**時的自動清理，**無法**防止您主動劃掉卡片。若希望 Bilirec 在清理多任務後仍保持運行，通常需要**鎖定多任務卡片**（各品牌操作見下方對應標籤頁）。

## 如何設定

請先完成 Bilirec App 內的**電池優化豁免**，再切換到您手機對應的系統標籤頁，按順序**逐步**調整。

<Aside type="tip">
**不必一次做完所有步驟。** 每完成一步，先測試 Bilirec 是否已恢復正常；若問題已解決，**即可停止，無需繼續往下做**。這樣可以在解決問題的同時，儘量減少對手機省電策略的影響。
</Aside>

<Tabs>
<TabItem label="華為 / 鴻蒙">

### 第一步：關閉「應用自動管理」

主要解決：App 切到後台或重啟後服務無法自啟、被系統清理。

1. 打開手機 **設定**，點擊 **電池**。
2. 點擊 **應用啟動管理**。
3. 在列表中找到 **Bilirec**，關閉其右側的 **自動管理** 開關。
4. 在彈出的手動管理窗口中，**同時勾選** 以下三項：
   - **允許自啟動**
   - **允許關聯啟動**
   - **允許後台活動**
5. 點擊 **確定** 保存。

完成後請測試 Bilirec。若後台運行已正常，無需繼續下一步。

### 第二步：加入電池優化白名單

主要解決：系統省電策略仍限制 Bilirec 後台活動。

1. 打開手機 **設定**，在頂部搜索欄輸入 **電池優化** 並進入。
2. 點擊頂部的下拉菜單，將 **不允許** 切換為 **所有應用**。
3. 在列表中找到 **Bilirec**，將其設為 **不允許**（即：不允許系統對本應用進行省電優化）。

<Aside type="tip">
不同系統版本的文案可能略有差異。關鍵是讓 Bilirec **不受電池優化**，與 App 內彈窗提示的目標一致。
</Aside>

完成後請測試 Bilirec。若問題已解決，無需繼續下一步。

### 第三步：休眠時保持網路連接

主要解決：鎖屏或手機休眠後網路被切斷，導致通訊中斷或超時。

1. 打開手機 **設定**，點擊 **電池**。
2. 點擊 **更多電池設定**。
3. 找到 **休眠時始終保持網路連接**（部分版本顯示為 **休眠時保持網路連接**），將其 **開啟**。

完成後請測試 Bilirec（建議鎖屏等待幾分鐘後再觀察）。若問題已解決，無需繼續下一步。

### 第四步：在多任務介面鎖定應用

主要解決：從多任務列表一鍵清理時 Bilirec 被連帶殺掉。

1. 從屏幕底部向上滑動並停頓，進入 **多任務管理介面**。
2. 找到 Bilirec 的卡片，用手指按住卡片並 **向下滑動**。
3. 當卡片右上角出現 **小鎖頭** 圖標時，表示後台鎖定成功。

### 使用移動數據時的額外說明

若在外出時使用 **移動數據**（5G / 4G）仍遇到連接卡頓、請求超時，可能是華為在使用移動數據時對後台進程施加了更嚴格的限制。可暫時連接 Wi‑Fi 或熱點；在 Wi‑Fi 下限制通常會明顯放寬。

若前幾步已解決問題，其餘步驟可不做；若仍有超時，再按順序嘗試後續步驟，或暫時改用 Wi‑Fi。

</TabItem>
<TabItem label="OPPO / ColorOS">

適用於大陸版 **OPPO**、**realme**、**OnePlus**（ColorOS 系統）。

### 第一步：關閉「應用自動管理」

主要解決：App 切到後台或重啟後服務無法自啟、被系統清理。

1. 打開手機 **設定**，點擊 **應用**。
2. 點擊 **應用啟動管理**。
3. 找到 **Bilirec**，改為 **手動管理**。
4. 勾選 **允許自啟動** 和 **允許後台活動**。

完成後請測試 Bilirec。若後台運行已正常，無需繼續下一步。

### 第二步：加入電池優化白名單

主要解決：系統省電策略仍限制 Bilirec 後台活動。

1. 打開手機 **設定**，點擊 **電池**。
2. 點擊 **更多設定**。
3. 點擊 **應用耗電管理**。
4. 找到 **Bilirec**，將其設為 **允許完全後台行為**。

<Aside type="tip">
不同系統版本的文案可能略有差異。關鍵是讓 Bilirec **不受省電限制**，與 App 內彈窗提示的目標一致。
</Aside>

完成後請測試 Bilirec。若問題已解決，無需繼續下一步。

### 第三步：關閉休眠時的系統自動清理

主要解決：夜間或熄屏後系統強制清理後台，導致通訊中斷或服務被終止。

1. 打開手機 **設定**，點擊 **電池**。
2. 點擊 **更多設定**。
3. 關閉 **睡眠模式優化** 或 **夜間超長續航**（視系統版本顯示項而定）。

完成後請測試 Bilirec（建議鎖屏等待幾分鐘後再觀察）。若問題已解決，無需繼續下一步。

### 第四步：在多任務介面鎖定應用

主要解決：從多任務列表劃掉卡片時 Bilirec 被立刻終止。

1. 從屏幕底部向上滑動並停頓，進入 **多任務管理介面**。
2. 找到 Bilirec 的卡片，點擊卡片右上角的 **⋮**，或 **長按** 卡片。
3. 選擇 **鎖定**。鎖定後，即使點擊「全清除」或手動上劃，該卡片也會留在後台不被關閉。

多數小米 / Redmi 用戶在完成 App 內電池優化豁免後，Bilirec 即可正常運行，**通常無需**額外調整系統設定。僅當您遇到後台被殺、鎖屏後服務中斷等狀況時，再參考下方指引排查。

- **省電策略** / **無限制**
- **後台運行**
