bilirec 的 Web 管理介面由 [bilirec-web](https://github.com/bilirec/bilirec-web) 提供，線上地址為 [app.bilirec.org](https://app.bilirec.org/)。它是一個通用前端，可連接任何 bilirec 後端實例。

## 訪問 Web 介面

1. 確保 bilirec 後端已啟動（默認監聽 `http://localhost:8080`）
2. 打開 [app.bilirec.org](https://app.bilirec.org/)

3. 在登錄介面填入後端地址（本地使用可留空或填 `http://localhost:8080`）
4. 若設置了 `USERNAME` 和 `PASSWORD`，輸入賬號密碼登錄；未設置則直接進入

## B 站登錄

進入主介面後，點擊「登入」，再點擊右上角頭像按鈕完成 B 站掃碼登錄。

## 連接自架後端

若後端不在本機運行，在登錄介面的**服務器地址**欄填入後端 URL，例如：

- 局域網：`http://192.168.1.100:8080`
- FRP 公網：`https://abc123.tunnel.bilirec.org`

## 安裝 PWA

bilirec-web 支持 PWA（Progressive Web App），可安裝到桌面或手機主屏幕，獲得接近原生 App 的體驗。

**桌面瀏覽器（Chrome / Edge）：**
- 地址欄右側點擊「安裝」圖標，或菜單 →「安裝 bilirec」

**手機瀏覽器：**
- iOS Safari：分享 →「添加到主屏幕」
- Android Chrome：菜單 →「安裝應用」或「添加到主屏幕」
