完成[安裝](/zh-tw/guides/installation/)後，按以下步驟啟動服務並開始使用。

## 啟動服務

<Tabs>
<TabItem label="Linux">

```bash
./bilirec-linux-amd64
```

首次運行二進制時，程式會在可執行檔案同目錄生成 `.env` 模板檔案，你可以直接編輯它修改設定。

</TabItem>

<TabItem label="Windows">

下載並運行 `bilirec-windows.exe`（雙擊或在 PowerShell 中執行）：

```powershell
.\bilirec-windows.exe
```

首次運行二進制時，程式會在可執行檔案同目錄生成 `.env` 模板檔案，你可以直接編輯它修改設定。

</TabItem>

<TabItem label="Docker">

```bash
docker pull eric1008818/bilirec:latest
docker run -d \
  --name bilirec \
  -p 8080:8080 \
  -v /path/to/records:/app/records \
  -v /path/to/secrets:/app/secrets \
  -v /path/to/database:/app/database \
  eric1008818/bilirec:latest
```

更多設定（環境變數、卷掛載說明、測試版本標籤）見 [Docker 指南](/zh-tw/guides/docker/)。

</TabItem>

<TabItem label="Android（bilirec-mobile）">

安裝 [bilirec-mobile](https://github.com/bilirec/bilirec-mobile/releases) 後，打開 App 點擊「啟動」即可先啟動後端服務。

詳細說明見 [Android App](/zh-tw/guides/android/)。

</TabItem>
</Tabs>

## 進入 Web 介面

<Tabs>
<TabItem label="Linux / Windows">

1. 打開 [app.bilirec.org](https://app.bilirec.org/)。
2. 在登入頁填入後端地址。由於瀏覽器安全限制，從 HTTPS 線上前端**只能**連接本機 HTTP 後端（`http://localhost:8080` 或 `http://127.0.0.1:8080`）；遠端後端須為 HTTPS（如 [FRP](/zh-tw/guides/frp/) 隧道）。
3. 若設定了 `USERNAME` / `PASSWORD`，先完成帳號登入。

</TabItem>

<TabItem label="Android（bilirec-mobile）">

1. 在 bilirec-mobile 內點擊「打開錄製程式」進入 PWA。
2. 如未自動跳轉，也可手動打開 [app.bilirec.org](https://app.bilirec.org/)。
3. 一般情況下後端地址已是 `http://127.0.0.1:8080`，可直接繼續下一步。

</TabItem>
</Tabs>

## B 站登入模式

<Tabs>

<TabItem label="controller（預設）">

服務啟動後，打開 [app.bilirec.org](https://app.bilirec.org/)（或你設定的 `FRONTEND_URL`），點擊「登入」，再到右上角點擊頭像按鈕完成 B 站掃碼登入。

```bash
export BILIBILI_LOGIN_MODE=controller # 默認已經是
```

<Tabs>
<TabItem label="桌面端">

![桌面端B站掃碼登入介面](/images/web-ui/desktop-b-qrcode.png)

</TabItem>
<TabItem label="移動端">

![移動端B站掃碼登入介面](/images/web-ui/mobile-b-qrcode.png)

</TabItem>
</Tabs>

</TabItem>

<TabItem label="startup">

啟動時在終端顯示二維碼，用手機 B 站 App 掃碼登入：

```bash
export BILIBILI_LOGIN_MODE=startup
```

</TabItem>

<TabItem label="anonymous">

直接匿名登入，無需掃碼。未登入可能無法錄製 1080p 及以上畫質：

```bash
export BILIBILI_LOGIN_MODE=anonymous
```

</TabItem>

</Tabs>

## 使用方式

<Tabs>
<TabItem label="Web 介面">

1. 存取 [app.bilirec.org](https://app.bilirec.org/) 並連接後端。
2. 在「錄製管理」裡手動開始/停止錄製。
3. 在「訂閱管理」裡添加房間並設定自動錄製。
4. 在「錄製檔案」裡瀏覽、下載或預覽錄製結果。

![桌面端登入介面](/images/web-ui/desktop-login.png)
![桌面端錄製管理](/images/web-ui/desktop-recording.png)

完整說明見 [Web 介面（PWA）](/zh-tw/guides/web-ui/)。

</TabItem>

<TabItem label="API">

可通過 REST API 直接接入腳本或自動化流程：

```bash
# 啟動錄製
curl -X POST "http://127.0.0.1:8080/record/12345/start"

# 停止錄製
curl -X POST "http://127.0.0.1:8080/record/12345/stop"
```

接口總覽見 [API 概述](/zh-tw/api/overview/)。

</TabItem>
</Tabs>

## 設定檔案載入順序

1. 當前工作目錄下的 `.env.local`（若存在，優先）
2. 可執行檔案同目錄的 `.env`（首次運行自動生成）
3. 環境變數（覆蓋檔案中的值）
4. Docker 環境：僅使用環境變數，不載入 `.env`

詳見[設定概述](/zh-tw/configuration/overview/)。
