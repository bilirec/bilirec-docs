完成[安裝](/zh-tw/guides/installation/)後，按以下步驟啟動服務並開始使用。

## 啟動服務

```bash
export BILIBILI_LOGIN_MODE=controller
export PORT=8080
export FRONTEND_URL=http://localhost:8080
./bilirec-linux-amd64
```

首次運行時，程序會在可執行檔案同目錄生成 `.env` 模板檔案，你可以直接編輯它修改設定。

export BILIBILI_LOGIN_MODE=anonymous

## 設定檔案加載順序

1. 當前工作目錄下的 `.env.local`（若存在，優先）
2. 可執行檔案同目錄的 `.env`（首次運行自動生成）
3. 環境變量（覆蓋檔案中的值）
4. Docker 環境：僅使用環境變量，不加載 `.env`

詳見[設定概述](/zh-tw/configuration/overview/)。
