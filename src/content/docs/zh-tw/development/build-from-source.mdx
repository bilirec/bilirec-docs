## 開發模式

```bash
make dev    # go run ./cmd/backend
```

## 構建二進制

```bash
make build  # 輸出 ./dist/bilirec-windows.exe（Windows）
```

支持平臺：Linux amd64/arm64、Windows、Android (.so)。

## Android 庫

需要 Android NDK 24+ 及 `./lib/<abi>/libffmpegkit.so`：

```bash
# Linux 主機
make android os=linux

# Windows 主機
make android os=windows
```

`NDK_HOME` 從 `ANDROID_NDK_LATEST_HOME`、`ANDROID_NDK_ROOT` 或 `ANDROID_NDK_HOME` 中選取。

輸出：`dist/android/<abi>/libbilirec.so`（默認 `arm64-v8a` 與 `x86_64`）。

## 要求

- Go 1.25+
- FFmpeg + ffprobe（可選，本地 MP4 轉換）
- Android NDK r24+（Android 構建）

預編譯二進制見 [GitHub Releases](https://github.com/bilirec/bilirec/releases)。
