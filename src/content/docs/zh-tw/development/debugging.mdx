## 啟用除錯

```bash
export DEBUG=true
```

伺服器啟動時會在日誌中打印一個臨時十六進制令牌。

## pprof 性能分析

除錯模式下在 `/debug/pprof` 掛載 pprof。該路由受保護：需在請求頭 `Authorization` 中填入啟動日誌中的十六進制令牌。

實現位於 `internal/modules/rest/rest.go`。

## 本地開發

```bash
make dev    # go run ./cmd/backend
```

詳見[從源碼建置](/zh-tw/development/build-from-source/)。
