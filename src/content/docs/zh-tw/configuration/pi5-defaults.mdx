---
title: 樹莓派 5B 默認設定
description: 針對樹莓派 5B + microSD 的默認優化參數
---

Bilirec 當前默認設定已針對**樹莓派 5B + microSD** 場景優化，在「降低寫入次數」和「控制內存佔用」之間取平衡。

| 優化項 | 值 | 說明 |
| ----- | ---- | ---- |
| `LIVE_STREAM_WRITER_SYNC_PERIOD_SECS` | `0` | 禁用週期性 fsync，僅在錄製結束時 sync。權衡：意外斷電可能丟失最後一段數據 |
| `LIVE_STREAM_WRITER_BUFFER_SIZE` | 8 MB | 降低 flush 頻率。@1080p30fps 約每 14 秒觸發一次滿緩衝寫入 |
| `LIVE_STREAM_WRITER_FLUSH_PERIOD_SECS` | `10` | 10 秒 flush，優先降低 SD 卡磨損 |
| `LIVE_STREAM_WRITER_CHAN_BUFFER_SIZE` | `64` | 單錄製約 32MB 隊列數據，適合 4GB 設備 |
| `SKIP_SMALL_FLUSH` | `true` | 低比特率流跳過小塊 flush |
| `LIVE_STREAM_WRITER_BYTES_POOL_SIZE` | 512 KB | 與常見 stream chunk 大小一致 |
| `SEQUENTIAL_WRITE` | `true` | 序列化多路寫入，保護 microSD |
| `MAX_CONCURRENT_RECORDINGS` | `3` | 保守併發上限，限制 RAM 峰值 |

容器運行時同步：`GOMEMLIMIT=768MiB`、`GOGC=100`。

若你的存儲介質是 SSD 或 HDD/NAS，請參閱對應的[調優方案](/zh-tw/configuration/ssd-tuning/)。
