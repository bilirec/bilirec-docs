## 基本轉換

| 變量 | 說明 | 默認值 |
|------|------|--------|
| `CONVERT_TO_MP4` | 錄製完成後轉為 MP4 | `false` |
| `DELETE_SOURCE_AFTER_CONVERT` | 轉換後刪除原始源檔案 | `false` |
| `NO_CONVERT_IF_INVALID` | 源檔案缺少音視頻流時停止轉碼 | `false` |

別名：`CONVERT_FLV_TO_MP4`、`DELETE_FLV_AFTER_CONVERT`。

啟用 `CONVERT_TO_MP4` 後，錄製完成的 FLV/TS 檔案自動加入轉換隊列，由後臺異步轉換為 MP4。

## 本地 FFmpeg

| 變量 | 說明 | 默認值 |
|------|------|--------|
| `FFMPEG_CHECK_INTERVAL_SECS` | 轉換任務輪詢間隔（秒） | `60` |
| `FFMPEG_MAX_CONCURRENT_TASKS` | 最大併發轉換任務數 | `1` |
| `FFMPEG_ALLOW_DURING_RECORDING` | 錄製中是否允許轉換 | `false` |
| `FFMPEG_ALLOW_DURING_RECORDING_MAX_ACTIVE_RECORDINGS` | 允許轉換的最大活躍錄製數 | `1` |

Docker 鏡像內置 FFmpeg 8.1.1。桌面環境需在 PATH 中安裝 FFmpeg 和 ffprobe。

## CloudConvert 雲端

| 變量 | 說明 | 默認值 |
|------|------|--------|
| `CLOUDCONVERT_API_KEY` | API Key（為空則禁用） | (未設置) |
| `CLOUDCONVERT_THRESHOLD` | 使用 CloudConvert 的檔案大小閾值（字節） | `1073741824` (1 GB) |
| `CLOUDCONVERT_CHECK_INTERVAL_SECS` | 任務狀態輪詢間隔（秒） | `180` |
| `CLOUDCONVERT_MAX_CONCURRENT_DOWNLOADS` | 最大併發下載數 | `1` |

檔案大小 ≥ 閾值時優先使用 CloudConvert。`PUBLIC_BASE_URL` 必須有效，否則 CloudConvert 會拒絕創建任務。
