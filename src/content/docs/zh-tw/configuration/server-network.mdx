## 服務器綁定

| 變量 | 說明 | 默認值 |
|------|------|--------|
| `HOST` | API 綁定地址，空值監聽所有網卡 | (空) |
| `PORT` | API 服務端口 | `8080` |

```bash
export HOST=0.0.0.0    # 監聽所有網卡
export PORT=8080
```

## HTTPS

同時設置證書和私鑰時，Fiber 默認啟用 HTTPS：

```bash
export SERVER_CRT=/path/to/server.crt
export SERVER_KEY=/path/to/server.key
```

<Aside type="caution">
啟用 FRP 且 `FRP_HTTPS=true` 時，必須同時設置 `SERVER_CRT` + `SERVER_KEY`，否則協議不匹配導致 FRP 無法工作。詳見 [FRP 設定](/zh-tw/configuration/frp/)。
</Aside>

## 反向代理

| 變量 | 說明 | 默認值 |
|------|------|--------|
| `TRUSTED_PROXIES` | 受信任代理 IP/CIDR（逗號分隔） | `161.33.159.26` |
| `PUBLIC_BASE_URL` | 後端公開基址（生成預簽名 URL） | (空) |
| `FRONTEND_URL` | 前端 URL（CORS） | `http://localhost:8080` |

`TRUSTED_PROXIES` 默認值 `161.33.159.26` 為公共 FRP 服務 IP。使用 Nginx 等反向代理時，將代理 IP 加入此列表。

`PUBLIC_BASE_URL` 必須包含 `http://` 或 `https://` 前綴，為空時預簽名 URL 回退為相對地址。
