bilirec 提供完整的 RESTful API，供程式調用、腳本控制或自行編寫管理介面。在 Android 上嵌入後端時，可先通過 [Android 庫接口](/zh-tw/development/android-library/) 啟動服務，再調用本文檔中的 HTTP API（或直接使用 [bilirec-web](https://app.bilirec.org/) 等現成前端）。

## Swagger UI

伺服器運行時，在根路徑 `/` 提供 Swagger UI — 在瀏覽器中打開 `http://your-host:8080/` 即可查看與測試所有 API。

## 認證

若設定了 `USERNAME` 與 `PASSWORD`，API 啟用基於 JWT 的認證。詳見 [認證 API](/zh-tw/api/auth/)。

未設定用戶名/密碼時，API 預設為公開存取。

## 通用約定

- 基礎 URL：`http://host:port`（預設 `http://localhost:8080`）
- 需要認證時，先 `POST /login` 獲取 `jwtToken` cookie，後續請求攜帶該 cookie
- 錯誤響應通常為 JSON：`{"error": "描述信息"}`
