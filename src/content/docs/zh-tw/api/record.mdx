## 開始錄製

```http
POST /record/:roomID/start?duration_minutes=<N>&stream_profile=<profile>&qn=<quality>&only_audio=<bool>
```

### duration_minutes

| 值 | 說明 |
| -- | ---- |
| 不傳 / `0` | 使用系統預設（`MAX_RECORDING_HOURS`） |
| `-1` | 無限錄製 |
| 正整數 | N 分鐘後自動停止 |

### stream_profile

允許的流格式清單（嚴格限制）。可傳單個值，或多個值用英文逗號分隔（如 `hls-fmp4,hls-ts`）。房間設定請改用 JSON 數組字段 `stream_profiles`。

| 值 | 說明 |
| -- | ---- |
| 不傳 / 空 | 允許全部格式；同畫質下按 HTTP-FLV → HLS-fMP4 → HLS-TS 排序嘗試 |
| `http-flv` | 僅 HTTP-FLV |
| `hls-ts` | 僅 HLS-TS |
| `hls-fmp4` | 僅 HLS-fMP4 |
| `hls-fmp4,hls-ts` | 僅允許所列格式（示例） |

多選時只會在所選格式內嘗試；同畫質下仍按 FLV → fMP4 → TS 排序。

### qn（畫質碼）

優先畫質（請求偏好，不保證與所選完全一致）。

| 值 | 畫質 |
| -- | ---- |
| 不傳 | 原畫（`10000`） |
| `80` | 流暢 |
| `150` | 高清 |
| `250` | 超清 |
| `400` | 藍光 |
| `10000` | 原畫 |
| `20000` | 4K |
| `30000` | 杜比 |

### only_audio

| 值 | 說明 |
| -- | ---- |
| 不傳 / `false` | 錄製音影片 |
| `true` | 僅錄製音頻 |

**注意：**

- 錄 4K / 杜比建議顯式傳 `qn`（如 `qn=20000`、`qn=30000`）
- `qn ≥ 20000` 時自動啟用 `*_HIGH` 高畫質記憶體池，一般無需改環境變數
- 若遇卡頓或記憶體不足，參見 [高碼率 / 4K 調參](/zh-tw/configuration/high-bitrate-tuning/) 與 [記憶體佔用估算](/zh-tw/configuration/memory-estimation/)

```bash
curl -X POST "http://127.0.0.1:8080/record/12345/start?duration_minutes=30&stream_profile=http-flv&qn=150&only_audio=true"
```

### 錯誤響應

| HTTP | 說明 |
| ---- | ---- |
| `400` | 無效參數、房間未開播、已在錄製、正在恢復流等 |
| `409` | 該房間錄製正在啟動中（並發請求衝突） |
| `410` | 無法連接到任何可用流 URL |
| `429` | 已達到 `MAX_CONCURRENT_RECORDINGS` 上限 |
| `507` | 磁碟可用空間低於 `MIN_DISK_SPACE_BYTES` |

常見 `400` 消息：`該房間已在錄製中`、`該房間正在恢復流`、`房間並非直播狀態`、`無可用的影片流 URL`。

## 停止錄製

```http
POST /record/:roomID/stop
```

## 批量獲取狀態

```http
POST /record/statuses
Content-Type: application/json

{ "roomIDs": [123, 456, 789] }
```

也支持 query：`?roomIDs=123,456,789`

```json
{ "123": "recording", "456": "idle", "789": "recovering" }
```

狀態值：`idle`（未錄製）、`recording`（錄製中）、`recovering`（流中斷，正在重連恢復）。

## 批量獲取統計

```http
POST /record/stats
Content-Type: application/json

{ "roomIDs": [123, 456, 789] }
```

```json
{
  "123": {
    "bytes_written": 1048576,
    "status": "recording",
    "start_time": 1234567890,
    "elapsed_seconds": 120,
    "output_path": "records/tester-123/live-title.flv",
    "room_title": "開錄時的直播標題"
  }
}
```

`room_title` 為**開始錄製時**固定的房間標題（用於輸出檔案名），與房間信息接口返回的當前標題可能不同。
